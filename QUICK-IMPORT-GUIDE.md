# 🎨 GUIDE RAPIDE - IMPORT COULEURS CSS

## ⚡ ACTION IMMÉDIATE

**Votre fichier est prêt :** `css-colors-import.csv` (148 couleurs)

### 🚀 ÉTAPES SIMPLES

1. **Ouvrez :** https://f6d72e-0f.myshopify.com/admin/settings/custom_data

2. **Créez le metaobject :**
   - Cliquez **"Add definition"**
   - **Name:** `Color Variant`
   - **Type:** `color_variant`
   - **Champs à ajouter :**
     - `name` (Single line text) ✅ Required
     - `hex_code` (Single line text) ✅ Required
     - `css_name` (Single line text) ✅ Required

3. **Sauvegardez** le type

4. **Import CSV :**
   - Retournez à la liste des metaobjects
   - Cliquez **"Import"**
   - Sélectionnez `css-colors-import.csv`
   - **Mappez les colonnes :**
     - `Name` → `name`
     - `Hex` → `hex_code` 
     - `Code` → `css_name`

5. **Confirmez** l'import
